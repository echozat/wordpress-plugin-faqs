# Project Structure

- _docs - the context & specs about the project
- assets - all the scss & js for the project separated as admin & public. After building the output is sent to build folder
- build - the compiled css & js file is stored here. admin.css, admin.js and public.css, public.js
- src - the complete codebase of the plugin
- src/core - all the core files of the plugin like PluginSetup is present in this folder
- src/admin - all admin views & classes are present here
- src/public - all public views & classes are present here
